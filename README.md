# API-STORE


## Features

- Get Products 
- Create Products 
- Delete Products 
- Update Products 
- Login and register user
- User Roles




-

## Installation


Install the dependencies and devDependencies and start the server.

```sh
cd API-STORE
npm i
node app
```

For production environments...

```sh
npm install --production
NODE_ENV=production node app
```
